+ chọn
    B1 git add -A
    B2 git status : kiểm tra
    B3 git commit -m "noi dung" :
    git log : kiểm tra sữa gần nhất lúc nào
+ Xác nhận
    git commit -m "noi dung commit"
+ Đẩy: lần đầu ~ git push -u origin master
